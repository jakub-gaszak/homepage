Notatki z WTF Co Ten Frontend

Nauka HTML-a

Z HTML-em spotykam się po raz pierwszy. Maciek w lekcji tłumaczy, że jest to język znaczników i nie należy postrzegać go jako języka programowania. HTML odpowiada za treść (a właściwie za jej opakowanie).

Style CSS

Kolejna lekcja dotyczyła stylów CSS. CSS odpowiada za styl czyli wygląd. Można zmienić kolor fontu, nadać tło itp.
